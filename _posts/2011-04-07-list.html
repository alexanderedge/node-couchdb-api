--- title: List Function categories: [api, designdocument] layout: api slug: list ---

Returns the contents of the specified list function. (not a rendered list of documents)

In order to get the results of a list function on a specific document, use the View API.

Also acts as a setter if you pass the contents of the list function as the second parameter.

NOTE: You must use the save() method in order to send your changes to the database.

{% highlight javascript %} var list = ddoc.list("my-list-function"); // list => "function (head, req) { ... }" // creating a new list function (or replacing an existing one) ddoc.list("my-list-function", function (head, req) { return "output"; }); // need to save these view views to the database ddoc.save(function (err, response) { // your new views are now available... }); {% endhighlight %}
