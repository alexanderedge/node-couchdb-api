Queries an arbitrary view, allows for querystring parameters via the first argument.

Protip: If you want to specifically get the results of either the map or reduce function for a view, use the map and reduce methods.

{% highlight javascript %} // specify querystring parameters (optional) var query = { include_docs: true }; view.query(query, function (err, response) { // your code here... }); {% endhighlight %}
