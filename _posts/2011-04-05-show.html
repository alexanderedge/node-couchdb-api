--- title: Show Function categories: [api, designdocument] layout: api slug: show ---

Returns the contents of the specified show function. (not a rendered document)

In order to get the results of a show function on a specific document, use the Document API.

Also acts as a setter if you pass the contents of the show function as the second parameter.

NOTE: You must use the save() method in order to send your changes to the database.

{% highlight javascript %} var show = ddoc.show("my-show-function"); // show => "function (doc, req) { ... }" // creating a new list function (or replacing an existing one) ddoc.show("my-show-function", function (doc, req) { return "output"; }); // need to save these view views to the database ddoc.save(function (err, response) { // your new views are now available... }); {% endhighlight %}
