Replicate a database to another database.

Additional Arguments *

name data type comments
source string|Database The source database; can be a string name, a string url or a Database object.
target string|Database The target database; can be a string name, a string url or a Database object.
opts * object Optional Parameters for the replication command
(see CouchDB documentation for specifics)

* callback is always assumed to be the last argument. See this page for more details regarding callbacks.

{% highlight javascript %} // using named (local) databases, replicate /test1 -> /test2 server.replicate("test1", "test2", function (err, response) { // your code here... }); // using db objects, replicate /test -> remote.com:5984/test var local = server.db("test"), remote = couchdb.srv("remote.com").db("test"); server.replicate(local, remote, function (err, response) { // your code here... }); // other options server.replicate("test1", "test2", { create_target: true }, function (err, response) { // your code here... }); {% endhighlight %}
