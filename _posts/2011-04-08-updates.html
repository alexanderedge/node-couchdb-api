--- title: Update Handlers categories: [api, designdocument] layout: api slug: updates ---

Returns a list of list functions in this design document.

This is essentially shorthand for ddoc.body.updates. It also initializes the object if it isn't already there.

Protip: If you add functions to the returned object, they will be sent to database if you call ddoc.save().

{% highlight javascript %} var updates = ddoc.updates(); // updates => { // test: "function (doc, req) { ... }" // } {% endhighlight %}
