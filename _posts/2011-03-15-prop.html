--- title: Property categories: [api, document] layout: api slug: prop ---

Shorthand for getting/setting a shallow property on a Document.

NOTE: This is chainable if you are setting a property, not if you are getting a property.

ANOTHER NOTE: If you are wanting to remove a property or set a deeper property, use the "long-hand" method. (doc.body)

{% highlight javascript %} doc.prop("foo", "bar"); // same as doc.body.foo = "bar"; doc.prop("foo") // same as doc.body.foo {% endhighlight %}
