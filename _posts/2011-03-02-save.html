Will either create a new document (if it doesn't already exist) or it will update the existing document. The existence is determined by the presence of an `_id` attribute.

{% highlight javascript %} doc.body.foo = "bar"; doc.body.hello = "world"; doc.save(function (err, response) { // your code here... }); {% endhighlight %}
