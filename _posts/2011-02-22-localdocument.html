--- title: Local Document categories: [api, database] layout: api slug: localDoc ---

Retrieves a LocalDocument object within the database's namespace.

NOTE: There is no API difference between LocalDocument and Document. The only difference is that _local/ is prepended to the name supplied for storage and retrieval.

{% highlight javascript %} var ldoc = db.localDoc("my-local-doc"); // or use the shorter alias: ldoc var ldoc = db.ldoc("my-local-doc"); {% endhighlight %}
