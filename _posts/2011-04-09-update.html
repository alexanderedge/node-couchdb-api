--- title: Update Handler categories: [api, designdocument] layout: api slug: update ---

Returns the contents of the specified update handler function.

Also acts as a setter if you pass the contents of the update handler as the second parameter.

NOTE: You must use the save() method in order to send your changes to the database.

{% highlight javascript %} var update = ddoc.update("my-update-function"); // update => "function (doc, req) { ... }" // creating a new update handler (or replacing an existing one) ddoc.update("my-update-function", function (doc, req) { return [doc, "message"]; }); // need to save these view views to the database ddoc.save(function (err, response) { // your new views are now available... }); {% endhighlight %}
