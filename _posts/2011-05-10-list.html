Executes a _list function on the current view.

Additional Arguments *

name data type comments
list string The list function name, using this view's design document. To override this, use the ddoc/list format.
query * object Optional Querystring parameters

* callback is always assumed to be the last argument. See this page for more details regarding callbacks.
