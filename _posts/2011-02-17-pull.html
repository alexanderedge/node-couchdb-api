Pull replicate another database into the current database. (basically swaps source/target from db.replicate / db.push)

Additional Arguments *

name data type comments
source string|Database The source database; can be a string name, a string url or a Database object.
opts * object Optional Parameters for the replication command
(see CouchDB documentation for specifics)

* callback is always assumed to be the last argument. See this page for more details regarding callbacks.
