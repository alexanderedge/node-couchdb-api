Executes a temporary view against the current database.

Additional Arguments *

name data type comments
map function The map function. Will be converted using .toString() before being sent to the server. (so closures will not apply)
reduce * function Optional The reduce function, see map function above.
query * object Optional Querystring parameters for the tempView command
(see CouchDB documentation for specifics)

* callback is always assumed to be the last argument. See this page for more details regarding callbacks.

{% highlight javascript %} // basic map function var map = function (doc) { emit(null, 1); }; // named reduce function var reduce = "_sum"; // querystring params var query = { include_docs: true }; db.tempView(map, reduce, query, function (err, response) { // your code here... }); {% endhighlight %}
