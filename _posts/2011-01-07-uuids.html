Retrieve a list of UUIDs generated by the server.

NOTE: The response from the server is being intercepted here and sending only the uuids property from the response, rather than the entire response.

Additional Arguments *

name data type default comments
count number 1 Number of UUIDs to request from the server (default: 1)

* callback is always assumed to be the last argument. See this page for more details regarding callbacks.

{% highlight javascript %} server.uuids(count, function (err, uuids) { // your code here... }); {% endhighlight %}
