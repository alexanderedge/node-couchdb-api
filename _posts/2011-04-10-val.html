--- title: Validation Function categories: [api, designdocument] layout: api slug: val ---

Returns the validate_doc_update function for this design document. This is essentially shorthand for ddoc.body.validate_doc_update

This function can also act as a setter by passing the contents of the validation function as the only parameter.

NOTE: You must use the save() method in order to send your changes to the database.

{% highlight javascript %} var val = ddoc.val(); // val => "function (newDoc, savedDoc, userCtx) { ... }" ddoc.val(function (newDoc, savedDoc, userCtx) { throw({ forbidden: "message" }); }); // need to save these view views to the database ddoc.save(function (err, response) { // your new views are now available... }); {% endhighlight %}
