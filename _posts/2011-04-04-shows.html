--- title: Show Functions categories: [api, designdocument] layout: api slug: shows ---

Returns a list of show functions in this design document.

This is essentially shorthand for ddoc.body.shows. It also initializes the object if it isn't already there.

Protip: If you add functions to the returned object, they will be sent to database if you call ddoc.save().

{% highlight javascript %} var shows = ddoc.shows(); // shows => { // test: "function (doc, req) { ... }" // } {% endhighlight %}
