--- title: List Functions categories: [api, designdocument] layout: api slug: lists ---

Returns a list of list functions in this design document.

This is essentially shorthand for ddoc.body.lists. It also initializes the object if it isn't already there.

Protip: If you add functions to the returned object, they will be sent to database if you call ddoc.save().

{% highlight javascript %} var lists = ddoc.lists(); // lists => { // test: "function (head, req) { ... }" // } {% endhighlight %}
