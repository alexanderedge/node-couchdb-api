node-couchdb-api The CouchDB API Wrapper for Node.js

Property Official Documentation


Method prop
Description Shorthand for getting/setting a shallow property on a Document. 
doc.prop("foo", "bar");
// same as
doc.body.foo = "bar";

doc.prop("foo")
// same as
doc.body.foo
