node-couchdb-api The CouchDB API Wrapper for Node.js

Show

GET /db/_design/design-doc/_show/show-name/doc Official Documentation

Will execute a _show function on the current document

Additional args

Name Type Default Comments
show string Must be in `ddoc/show` format
query Optional hash Query string parameters
(See Official Documentation) 
doc.show("ddoc/show", { format: "html" }, function (err, response) {
   // your code here...
});
