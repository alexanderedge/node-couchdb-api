node-couchdb-api The CouchDB API Wrapper for Node.js

Local Document API Official Documentation

var couchdb = require("couchdb-api");

/** default values: "localhost", 5984 **/
var server = couchdb.srv(host, port);

var db = server.db("my-database");

var doc = db.ldoc("my-local-doc");

LocalDocument objects inherit directly from Document objects. So far, the API is identical.
