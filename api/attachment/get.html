Method get
Description Returns the contents of the specified attachment
HTTP GET /db/doc/attachment

Arguments

Name Type Default Comments
stream Optional boolean false

If left out, (or falsy) entire response will be contsructed before running callback.

Always use streaming when you have large files.
callback function See this page for more information 
attachment.get(function (err, content) {
  // err     => errors (if any)
  // content => entire contents of file
});

attachment.get(true, function (err, stream) {
  // err    => errors (if any)
  // stream => ReadableStream of source attachment
})
