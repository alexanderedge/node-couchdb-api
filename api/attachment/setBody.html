Set Body Official Documentation


Method setBody
Description Sets the contents and mime type for the attachment

Arguments

Name Type Default Comments
format string

Determines the MIME type for the attachment. Some shortcuts are available, but anything outside that list will need to include the full MIME type.

Shortcuts:
text
text/plain
json
application/json
html
text/html
xml
application/xml
body string|Buffer|ReadableStream
callback function See this page for more information 
attachment.setBody("text", "This is my attachment");
attachment.setBody("html", new Buffer("<html><body>This is my HTML attachment</body></html>"));
attachment.setBody("image/png", fs.createReadStream("my-image-file.png"));

attachment.save(function (err, response) {
  // err      => errors (if any)
  // response => response from the server
})
