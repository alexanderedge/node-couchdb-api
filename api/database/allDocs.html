Method allDocs
Description Queries the database for all of it's documents
HTTP
  • GET /db/_all_docs
  • POST /db/_all_docs

Arguments

Name Type Default Comments
query hash Query-string arguments (same API as Views)
keys array An array of document ids, this is useful for bulk-selecting several documents with known ids without a view.
callback function See this page for more information 
// get all
db.allDocs(function (err, result) {
    // do stuff...
});

// specify document ids
db.allDocs(["my-doc-1", "my-doc-2"], function (err, result) {
    // do stuff...
});

// add query string params
db.allDocs({ include_docs: true }, function (err, result) {
    // do stuff...
});

// specify document ids and query string params
db.allDocs({ include_docs: true }, ["my-doc-1", "my-doc-2"], function (err, result) {
    // do stuff...
});
