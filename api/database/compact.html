Compact Official Documentation


Method compact
Description Run a compaction on the database itself, or the view indexes for a specified design document
HTTP POST /db/_compact POST /db/_compact/ddoc

Arguments

Name Type Default Comments
ddoc Optional string A specified design document to target instead
callback function See this page for more information 
// compact the whole database
db.compact(function (err, result) {
   // your code here...
});

// compact only this design document
db.compact("my-design-doc", function (err, result) {
   // your code here...
});
